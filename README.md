Projek ini berisikan Automatic Number Plate Recognition (ANPR) memakai YOLOv5, EasyOCR, CLAHE, dan Regular Expression (RegEx)
Kodingan py dan Model YOLO berada di folder Model
Model YOLO ada 2 yakni: 1. YOLO yang memakai CLAHE, dan 2. YOLO yang tidak memakai CLAHE
Cara Memakai: 1. Download YOLOv5 dari Github, 2. Pada kodingan py konfigurasi path gambar yang diinput dan output, 3. Jalankan kodingan
